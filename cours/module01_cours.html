01 if __name__ == "__main__" et le shebang

Quand Python exécute un fichier, il lui assigne la variable spéciale __name__. Si le fichier est lancé directement, __name__ vaut "__main__". Si le fichier est importé par un autre, __name__ vaut le nom du fichier.

# ft_garden_intro.py
name = "Rose"
height = 25
age = 30

if __name__ == "__main__":
    print("=== Welcome to My Garden ===")
    print(f"Plant: {name}")
    print(f"Height: {height}cm")
    print(f"Age: {age} days")
    print("=== End of Program ===")
Le shebang : la première ligne #!/usr/bin/env python3 permet d'exécuter le script directement (./ft_garden_intro.py) sans taper python3, après un chmod +x.

02 Classes — les bases

Une classe est un moule qui décrit des données (attributs) et des comportements (méthodes). Un objet est une instance de ce moule.

03 __init__ — le constructeur

__init__ est la méthode appelée automatiquement à la création d'un objet. self est la référence à l'objet lui-même — il doit être le premier paramètre de toute méthode d'instance.

class Plant:
    def __init__(self, name: str, height: float, age: int) -> None:
        self._name = name      # underscore = attribut "protégé"
        self._height = height
        self._age = age

    def show(self) -> None:
        print(f"{self._name}: {self._height}cm, {self._age} days old")

# Création d'instances :
rose = Plant("Rose", 25.0, 30)
cactus = Plant("Cactus", 15.0, 120)
rose.show()   # → Rose: 25.0cm, 30 days old

04 Méthodes d'instance — grow(), age()

class Plant:
    def __init__(self, name: str, height: float, age: int) -> None:
        self._name = name
        self._height = height
        self._age = age
        self._growth_rate: float = 0.8

    def grow(self) -> None:
        self._height = round(self._height + self._growth_rate, 1)

    def age(self) -> None:
        self._age += 1

    def show(self) -> None:
        print(f"{self._name}: {self._height}cm, {self._age} days old")

05 Encapsulation — attributs protégés et getters/setters

La convention _attribut (un underscore) signale un attribut "protégé". On n'y accède que via des getters/setters qui valident les données.

class Plant:
    def __init__(self, name: str, height: float, age: int) -> None:
        self._name = name
        self._height = 0.0
        self._age = 0
        self.set_height(height)
        self.set_age(age)

    def set_height(self, value: float) -> None:
        if value < 0:
            print(f"{self._name}: Error, height can't be negative")
            return
        self._height = value
        print(f"Height updated: {value}cm")

    def get_height(self) -> float:
        return self._height

06 Héritage et super()

super() donne accès à la classe parente. super().__init__() appelle le constructeur de Plant pour éviter de dupliquer le code.

class Flower(Plant):
    def __init__(self, name: str, height: float,
                 age: int, color: str) -> None:
        super().__init__(name, height, age)  # appelle Plant.__init__
        self._color = color
        self._blooming = False

    def bloom(self) -> None:
        self._blooming = True

    def show(self) -> None:
        super().show()                       # réutilise Plant.show()
        print(f" Color: {self._color}")
        if self._blooming:
            print(f" {self._name} is blooming beautifully!")
        else:
            print(f" {self._name} has not bloomed yet")

07 @staticmethod et @classmethod

class Plant:
    @staticmethod
    def is_older_than_year(age: int) -> bool:
        return age > 365

    @classmethod
    def anonymous(cls) -> "Plant":
        return cls("Unknown plant", 0.0, 0)

# Appel :
print(Plant.is_older_than_year(400))   # → True
p = Plant.anonymous()
p.show()   # → Unknown plant: 0.0cm, 0 days old

08 Classe imbriquée (nested class)

Chaque Plant possède un système de statistiques interne, implémenté comme une classe à l'intérieur de la classe principale.

class Plant:
    class _Stats:
        def __init__(self) -> None:
            self._grow_count: int = 0
            self._age_count: int = 0
            self._show_count: int = 0

        def display(self) -> None:
            print(f"Stats: {self._grow_count} grow, "
                  f"{self._age_count} age, "
                  f"{self._show_count} show")

    def __init__(self, name: str, height: float, age: int) -> None:
        self._name = name
        self._height = height
        self._age = age
        self._stats = Plant._Stats()

    def grow(self) -> None:
        self._height = round(self._height + 0.8, 1)
        self._stats._grow_count += 1

    def show(self) -> None:
        print(f"{self._name}: {self._height}cm, {self._age} days old")
        self._stats._show_count += 1

10 Erreurs fréquentes à éviter

1. Oublier self comme premier paramètre :
def show(): → TypeError
def show(self): → correct
2. Appeler une méthode sans parenthèses :
rose.show → référence, ne fait rien
rose.show() → appel correct
3. Oublier super().__init__() dans une sous-classe :
Sans cet appel, les attributs de la classe parente (_name, _height, _age) n'existent pas sur l'objet.
4. Confondre attribut de classe et d'instance :
count = 0 au niveau de la classe → partagé par toutes les instances
self._height = 0 dans __init__ → propre à chaque objet
Convention de nommage :
Classes → PascalCase : Plant, Flower, GardenManager
Méthodes et variables → snake_case : set_height, growth_rate
Attributs protégés → préfixe underscore : _name, _height
