01 print() — afficher du texte

print() envoie du texte vers la sortie standard (le terminal). C'est la fonction la plus simple et la première que tu utilises.

print("Hello, Garden Community!")   # affiche du texte
print("Récolte :", 42)              # affiche texte + nombre
print("a =", 3, "b =", 5)          # plusieurs arguments
print()                             # ligne vide

Comportement important : print() ajoute automatiquement un saut de ligne à la fin. Pour l'éviter : print("texte", end="").

02 input() — lire une saisie utilisateur

name = input("Enter garden name: ")  # affiche le prompt, attend la saisie
print("Garden:", name)
Attention cruciale : input() retourne toujours une chaîne de caractères (str). Pour travailler avec des nombres, il faut convertir :

age = int(input("Âge en jours : "))

06 Boucles — for et récursion

def ft_count_harvest_iterative():
    n = int(input("Days until harvest: "))
    for i in range(1, n + 1):
        print("Day", i)
    print("Harvest time!")

def ft_count_harvest_recursive():
    n = int(input("Days until harvest: "))
    def count(i):
        if i > n:
            print("Harvest time!")
            return
        print("Day", i)
        count(i + 1)
    count(1)

range(start, stop) génère les entiers de start à stop - 1. Pour aller jusqu'à n inclus : range(1, n+1).

07 Fonctions — def

09 Type hints (ex. 7)

def ft_seed_inventory(seed_type: str, quantity: int, unit: str) -> None:
    name = seed_type.capitalize()
    if unit == "packets":
        print(f"{name} seeds: {quantity} packets available")
    elif unit == "grams":
        print(f"{name} seeds: {quantity} grams total")
    elif unit == "area":
        print(f"{name} seeds: covers {quantity} square meters")
    else:
        print("Unknown unit type")

-> None signifie que la fonction ne retourne rien. Les types courants : str, int, float, bool, None. Vérifie avec mypy.

10 Erreurs fréquentes à éviter

1. Oublier de convertir input() :
days = input("Jours : ") → days est un str, pas un int !
Toujours faire : days = int(input("Jours : "))
2. Confondre = et == :
if age = 60: → SyntaxError
if age == 60: → correct
3. Mauvaise indentation :
Python utilise l'indentation (4 espaces) à la place des accolades. Un oubli = IndentationError.
4. range() : borne haute exclue :
range(1, 5) → 1, 2, 3, 4 (pas 5). Pour aller jusqu'à n : range(1, n+1).
5. Récursion sans cas de base :
Une fonction récursive sans condition d'arrêt boucle indéfiniment → RecursionError.
